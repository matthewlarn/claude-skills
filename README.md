# Claude Skills Repository

This repository contains a collection of specialized skills and tools for enhancing accessibility and auditing capabilities within Claude AI workflows.

## Folders

### accessibility-skills-complete/
A comprehensive set of accessibility-focused skills covering various aspects of web and digital accessibility, including:
- WCAG compliance checklists
- Alt text generation
- Keyboard navigation auditing
- Screen reader scripting
- Contrast checking
- And many more specialized accessibility tools

### cowork-agent-audit/
A framework for auditing and evaluating coworking agent systems, including:
- Accessibility audits
- Brand consistency checks
- Business strategy analysis
- Design criticism
- Frontend development reviews
- Performance optimization
- UX design evaluation

## Usage

These skills are designed to be integrated into Claude AI workflows to provide specialized expertise in accessibility and coworking agent auditing.

Contributions are welcome! Please ensure that new skills follow the established patterns and include appropriate documentation.
