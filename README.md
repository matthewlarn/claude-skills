# Claude Skills Repository

This repository collects reusable Claude skills for accessibility review, prototype critique, and related workflows.

## Current Collections

### `accessibility-skills-complete/`
A broad set of accessibility-focused skills for web and digital products, including:
- WCAG compliance checklists
- Alt text generation
- Keyboard navigation auditing
- Screen reader scripting
- Contrast checking
- AI transparency, privacy, and disability-focused audit helpers

### `cowork-agent-audit/`
A multi-perspective prototype review skill for evaluating design and implementation quality, including:
- Accessibility audits
- Brand and design consistency checks
- Business strategy analysis
- Design criticism
- Frontend development reviews
- Performance optimization
- UX design evaluation

### `cowork-skills/cowork-agent-audit/`
A namespaced copy of the same cowork audit skill package for users who want the skill grouped under `cowork-skills/`.

### `design-skills/`
Reserved for design-oriented skills. This folder exists in the repo, but it is not yet populated.

### `other/`
Reserved for miscellaneous skills. This folder exists in the repo, but it is not yet populated.

## Usage

These skills are intended to be dropped into Claude AI workflows when you need focused expertise for accessibility, design review, or related task-specific analysis.

## Contributing

Contributions are welcome. Please follow the existing folder conventions and include clear documentation for any new skills or skill packages.
